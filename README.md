💊 Conversor de Unidades Farmacêuticas

Programa de terminal para conversão de unidades farmacêuticas comuns.
Desenvolvido em Python puro, sem dependências externas.

## Funcionalidades

- Conversão entre mg, g e mcg
- Conversão entre mL e L
- Cálculo de concentração percentual (% m/v)
- Cálculo de dose total por peso do paciente (mg/kg × kg)

## Como usar

```bash
python conversor.py
```

## Tecnologias

- Python 3.x
- Nenhuma biblioteca externa
